# VCborn Fes Layouts

This is the NodeCG bundle for the VCborn Fes.

## Dependencies

* NodeCG
* twitter-api-v2
* youtube-chat
* swiper.js

## Installation

1. Setup NodeCG
2. Clone this repository to bundles/vcborn-fes
3. Install node packages
4. Create a config named "vcborn-fes.json"
